## 一、需求背景

用户对第一版Apple极简风格不满意，明确要求采用**苹果软件主流的Liquid Glass（液态玻璃）风格**。

Liquid Glass是Apple在iOS 26、macOS 26中引入的最新设计语言，核心特征：
- 🌊 毛玻璃效果（Glassmorphism）
- ✨ 半透明表面
- 🎨 渐变色彩
- 💫 流体动效
- 💎 光泽质感

---

## 二、设计决策

### 核心特征对比

| 特性 | 极简Apple | Liquid Glass |
|------|----------|--------------|
| **背景** | 纯白 #FFFFFF | 渐变 #F5F5F7→#E8E8ED |
| **表面** | 不透明 | 半透明 50-70% |
| **模糊** | 无 | blur(20-40px) |
| **色彩** | 单色 Apple Blue | 蓝紫粉橙渐变 |
| **圆角** | 10-12px | 14-32px |
| **动效** | 300-400ms | 500-1200ms |
| **装饰** | 无 | 液态球 + 渐变流动 |
| **字体** | 17px Regular | 19px Medium |

### 色彩系统

**主色渐变**：
```css
/* 蓝紫粉渐变 */
linear-gradient(135deg, #0071E3 0%, #5856D6 50%, #AF52DE 100%)

/* 青蓝渐变 */
linear-gradient(135deg, #5AC8FA 0%, #0071E3 100%)

/* 橙粉渐变 */
linear-gradient(135deg, #FF9500 0%, #FF2D55 100%)
```

**玻璃表面**：
- 标准玻璃白：`rgba(255, 255, 255, 0.7)` + `backdrop-filter: blur(30px)`
- 轻玻璃：`rgba(255, 255, 255, 0.5)` + `backdrop-filter: blur(20px)`

**动画**：
- 背景渐变流动：15秒循环
- 按钮渐变流动：6秒循环
- 装饰球漂浮：20-25秒循环

### 字体系统

**字号增大**：
- Large Title: 34px → **40px**
- Title 1: 28px → **32px**
- Title 2: 22px → **26px**
- Body: 17px → **19px**

**字重加粗**：
- 标题：700 → **800**（ExtraBold）
- 正文：400 → **500**（Medium）
- 不再使用Regular（400）

### Glassmorphism核心

```css
/* 标准毛玻璃卡片 */
.glass-card {
    background: rgba(255, 255, 255, 0.7);
    backdrop-filter: blur(30px) saturate(180%);
    border-radius: 24px;
    border: 1px solid rgba(255, 255, 255, 0.4);
    box-shadow:
        0 8px 32px rgba(0, 113, 227, 0.12),
        0 0 0 1px rgba(255, 255, 255, 0.5) inset;
}
```

### 装饰元素

**液态装饰球**（3个）：
- 大小：400-600px
- 位置：背景漂浮
- 模糊：`filter: blur(80px)`
- 透明度：0.25
- 动画：20-25秒漂浮循环

## 五、技术限制

### Streamlit的限制

由于Streamlit的CSS支持有限，无法完全实现浏览器级别的Liquid Glass效果：

**无法实现**：
- ❌ 装饰球（Streamlit不支持::before伪元素定位）
- ❌ 真正的多层玻璃叠加
- ❌ 复杂的3D变换

**成功实现**：
- ✅ 毛玻璃效果（backdrop-filter）
- ✅ 渐变背景和按钮
- ✅ 动画关键帧
- ✅ 超大圆角
- ✅ 弹性交互

### 浏览器兼容性

**需要**：
- Chrome 76+（backdrop-filter支持）
- Safari 9+（-webkit-前缀）
- Firefox 103+（backdrop-filter支持）

**不支持**：
- 旧版浏览器（<2019）

## 六、性能考虑

### 性能优化

1. **模糊强度限制** — 20-40px（不是100px）

## 八、设计原则总结

### ✅ DO

1. **大量使用毛玻璃** — backdrop-filter: blur(20-40px)
2. **半透明背景** — rgba(255, 255, 255, 0.5-0.7)
3. **渐变色彩** — 蓝紫粉橙流动渐变
4. **流体动效** — 500-1200ms慢速动画
5. **超大圆角** — 14-32px
6. **多层叠加** — 3-5层玻璃深度
7. **微妙光泽** — 内发光、高光、反射

### ❌ DON'T

1. **不使用纯色背景** — 必须渐变
2. **不使用不透明表面** — 必须半透明
3. **不使用小圆角** — 最小14px
4. **不使用快速动画** — 最小400ms
5. **不过度装饰** — 装饰球3个即可
