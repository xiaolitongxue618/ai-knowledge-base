### 2. 问答历史保存功能 💾

**需求来源：** 用户需要回顾之前的对话

**实现内容：**
- 创建数据库表 `chat_history`
- 每次问答自动保存到数据库
- 侧边栏显示最近10条历史
- 可展开查看完整问答内容
- 一键清空所有历史

**数据库设计：**
```sql
CREATE TABLE IF NOT EXISTS chat_history (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    question TEXT NOT NULL,
    answer TEXT NOT NULL,
    sources TEXT,  -- JSON 格式
    timestamp TIMESTAMP DEFAULT CURRENT_TIMESTAMP
)
```

**新增方法：**
- `save_chat()` - 保存问答记录
- `get_chat_history()` - 获取历史记录
- `clear_chat_history()` - 清空历史

**UI 展示：**
```python
# 侧边栏显示
st.markdown("### 📜 问答历史")
chat_history = metadata_store.get_chat_history(limit=10)

for chat in chat_history:
    with st.expander(f"Q: {chat['question'][:40]}..."):
        st.markdown(f"**问题:** {chat['question']}")
        st.markdown(f"**答案:** {chat['answer']}")
        st.caption(f"⏰ {chat['timestamp']}")
```

**效果：**
- ✅ 问答记录持久化
- ✅ 方便回顾历史对话
- ✅ 提升系统实用性

### 3. 文档搜索和过滤功能 🔍

**需求来源：** 文档多了之后查找困难

**实现内容：**
- 按文档名搜索（模糊匹配）
- 按文档类型过滤（PDF/MD/TXT）
- 按状态过滤（活跃/失败）
- 按时间/大小/名称排序
- 显示过滤结果数量

**技术实现：**
```python
# 搜索框
search_query = st.text_input("🔎 搜索文档名")

# 过滤器
filter_type = st.selectbox("📁 文档类型", ["全部", "pdf", "md", "txt"])
filter_status = st.selectbox("📊 状态", ["全部", "active", "failed"])
sort_by = st.selectbox("🔃 排序", ["时间", "大小", "名称"])

# 应用过滤
filtered_docs = docs.copy()
if search_query:
    filtered_docs = [d for d in filtered_docs if search_query.lower() in d.file_name.lower()]
if filter_type != "全部":
    filtered_docs = [d for d in filtered_docs if d.file_type == filter_type]
if filter_status != "全部":
    filtered_docs = [d for d in filtered_docs if d.status.value == filter_status]

# 排序
if sort_by == "大小":
    filtered_docs = sorted(filtered_docs, key=lambda x: x.file_size, reverse=True)
elif sort_by == "名称":
    filtered_docs = sorted(filtered_docs, key=lambda x: x.file_name)
```

**效果：**
- ✅ 快速找到目标文档
- ✅ 按需查看文档分类
- ✅ 提升管理效率

### 4. 导出功能 📤

**需求来源：** 用户需要保存和分享数据

**实现内容：**
- 导出问答记录（Markdown 格式）
- 导出文档列表（表格形式）
- 一键下载文件

**技术实现：**

**问答记录导出：**
```python
# 生成 Markdown 内容
md_content = "# 问答历史记录\n\n"
md_content += f"导出时间: {datetime.now()}\n\n"
md_content += f"总计: {len(chat_history)} 条记录\n\n"

for i, chat in enumerate(reversed(chat_history), 1):
    md_content += f"## 问题 {i}\n\n"
    md_content += f"**时间:** {chat['timestamp']}\n\n"
    md_content += f"**问题:** {chat['question']}\n\n"
    md_content += f"**答案:** {chat['answer']}\n\n"
    if chat.get('sources'):
        md_content += f"**来源:**\n"
        for source in chat['sources']:
            md_content += f"- {source['source_name']}\n"

# 提供下载
st.download_button(
    label="⬇️ 下载 Markdown 文件",
    data=md_content,
    file_name=f"chat_history_{datetime.now().strftime('%Y%m%d_%H%M%S')}.md",
    mime="text/markdown"
)
```

**文档列表导出：**
```python
# 生成 Markdown 表格
md_content = "# 文档列表\n\n"
md_content += "| 文件名 | 类型 | 大小 (KB) | 状态 | 块数 |\n"
md_content += "|--------|------|-----------|------|------|\n"

for doc in docs:
    md_content += f"| {doc.file_name} | {doc.file_type} | {doc.file_size / 1024:.1f} | {doc.status.value} | {doc.chunk_count} |\n"
```

**效果：**
- ✅ 数据持久化到本地
- ✅ 方便分享和备份
- ✅ 支持 Markdown 格式

## 💥 遇到的问题和解决方案

### 问题 1: Streamlit expander() 不支持 key 参数

**错误信息:**
```
TypeError: LayoutsMixin.expander() got an unexpected keyword argument 'key'
```

**原因:**
```python
# 错误写法
with st.expander(f"Q: {chat['question'][:40]}...", key=f"history_{i}"):
```

Streamlit 的 `st.expander()` 不支持 `key` 参数（与其他组件不同）。

**解决方案:**
```python
# 正确写法 - 移除 key 参数
for chat in chat_history:
    with st.expander(f"Q: {chat['question'][:40]}..."):
        # ...
```
